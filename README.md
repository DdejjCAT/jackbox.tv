# Jackbox.tv Mirror

Зеркало для игры Jackbox с возможностью запуска на собственном сервере.

## 🚀 Быстрая установка

Для работы зеркала нужен сервер + настройка домена. Вот самый быстрый способ:

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/DdejjCAT/jackbox.tv.git
cd jackbox.tv
```

### 2. Замените домен на свой

Редачьте
main/@moderator/C9ujNxqs.js
CNIlMMVr.js
script-0.js

### 3. Настройте веб-сервер

Настройте Nginx/Apache для обслуживания файлов из этой папки на вашем домене.

### 4. Настройте SSL (опционально)

Для работы через HTTPS используйте Let's Encrypt:

```bash
sudo certbot --nginx -d ВАШ ДОМЕН СЕРВЕРА
```

Эти файлы были скопированы вручную и могут требовать дополнительной настройки.
